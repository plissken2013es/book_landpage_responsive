Guía para crear un videojuego "indie" en 72 horas

El libro

Este manual cubre varios temas muy amplios de la manera más breve y accesible posible y con la filosofía central de ponerse manos a la obra. Se trata de hacer algo, conseguir acabarlo, disfrutar mientras lo hacemos y, si puede ser, aprender un montón de cosas por el camino.

Se trata de una guía general para desarrollar un videojuego con el motor de dos de las librerías HTML5 más populares de la red. Haremos un juego desde cero de forma que quede listo para una “Game Jam” o “Compo”, y lo que habremos aprendido cuando lleguemos a este punto nos servirá para crear otros juegos similares (y no tan parecidos).

Dos potentes librerías HTML5

Aprenderemos a usar dos de las librerías HTML5 de desarrollo de juegos 2D más potentes: ImpactJS y Phaser.

Ambas llevan activas el tiempo suficiente para lograr una robustez y estabilidad enormes. Además, disponen de una comunidad muy amplia de gente apasionada por los juegos, que puede resolver nuestras dudas y problemas cuando estos aparezcan.

Las dos nos proporcionan la posibilidad de desarrollar un videojuego una única vez y obtener una aplicación compatible con cualquier navegador web, que -además- se puede empaquetar como aplicación nativa para IOS o Android.
